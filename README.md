# Phỉ Thúy Lê Công (Jade Elegance) - E-commerce Platform

A modern, high-end e-commerce platform specializing in premium Jade gemstones (Phỉ Thúy). Built with a focus on visual excellence, smooth animations, and robust administrative management.

## 🌟 Features

### 🛍️ Customer Experience
- **Elegant Product Selection**: Categorized and filtered catalogs for high-quality Jade pieces.
- **Multilingual Support**: Full support for Vietnamese and English across all pages (Products, Blogs, Contact, etc.).
- **Smooth Navigation**: Powered by Framer Motion for smooth transitions and responsive design on all devices.
- **Knowledge Center**: A dedicated blog section (**Cẩm Nang & Kiến Thức**) for Jade educational content.
- **Secure Ordering**: Streamlined cart and checkout process with email notifications and bank QR payment support.

### 🛡️ Administrative Control
- **Dynamic Dashboard**: Centralized management for products, collections, and orders.
- **Advanced Content Management**:
  - **Rich Text Blog Editor**: Write and format educational content effortlessly using React Quill (WYSIWYG).
  - **Auto-Slug Generation**: SEO-friendly URL generation for products and blogs.
- **Business Configuration**:
  - **Payment Settings**: Manage bank details and upload QR codes for instant payments.
  - **Social & Contact Info**: Update social media links and store contact details dynamically.
- **Voucher & Promotions**: Create and manage localized discount codes and homepage banners.
- **Admin Management**: Secure multi-admin authentication with email credentials and password recovery.

## 🛠️ Technology Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend**: Node.js, Express, PostgreSQL.
- **Email**: Nodemailer for transactional emails.
- **Editor**: React-Quill-New (Rich Text formatting).

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Environment Configuration**:
   Configure your `.env` file with PostgreSQL credentials and SMTP settings.
3. **Run Dev Server**:
   ```bash
   npm run dev
   ```
4. **Production Build**:
   ```bash
   npm run build
   ```
